MacBook Pro 13" 2017 (A1708) Display Overview

The 2017 MacBook Pro 13" function key model offered the new thin design without Touch Bar. Model identifier A1708 covers both 2016 and 2017 two port models that retained traditional function keys instead of the Touch Bar strip. Two Thunderbolt 3 ports and a headphone jack provided connectivity. This entry level configuration appealed to users who wanted the updated design without Touch Bar premium or complexity.

Mac Screen Repair Sydney specialise in the aspects of Retina display repair for A1708 MacBook Pro models. The 13" display delivers 2560 x 1600 resolution with P3 colour support. We've worked on alot of these function key models which remain popular due to their straightforward design. Quality replacement parts ensure proper display performance after repair.

Screen Repair Options

Two repair methods address screen damage on the 13" 2017 function key model. LCD only replacement removes the damaged internal panel while preserving your original aluminium housing and camera. This requires housing without dents or structural damage. Full display assembly replacement swaps the entire top section when housing damage prevents LCD only repair.

The team at Mac Screen Repair Sydney evaluate devices before recommending either approach. We dont push assembly replacements when LCD only achieves equivalent results. Housing condition and damage extent determine our suggestion. Appropriate repair without unnecessary cost.

Display Technical Specifications

The A1708 features a 13.3-inch LED-backlit IPS Retina display at 2560 x 1600 native resolution. This provides 227 pixels per inch for sharp text and detailed images. The panel supports P3 wide colour gamut for accurate colour reproduction. Maximum brightness reaches 500 nits which handles typical lighting conditions adequately.

The function key model uses the same display panel specifications as the Touch Bar A1706 model. Both deliver identical screen quality. The A1708 simply lacks the Touch Bar strip and has two ports instead of four. Replacement panels match original colour accuracy and brightness specifications.

No True Tone Feature

The 2017 13" function key model does not include True Tone technology. This feature arrived with the 2018 models. For repair purposes, this simplifies things since theres no True Tone data to preserve. Both LCD only and full assembly replacement deliver identical display functionality. No features are lost regardless of repair method.

Flexgate Considerations

The 2016-2017 13" models including the A1708 can develop display cable wear over time. Apple's design routes the cable through the hinge where repeated opening creates stress. Symptoms include stage lighting at the bottom edge, flickering, or display blackouts at certain angles. The problem typically worsens if left unaddressed.

Apple extended warranty coverage for some affected units though many machines now fall outside that program. The team have repaired flexgate issues on A1708 models. Full display assembly replacement addresses cable problems since it includes new cables. We diagnose whether your symptoms indicate cable wear versus panel damage.

Frequently Asked Questions

Is the A1708 display same as A1706 Touch Bar?

The panel specifications are identical between A1708 and A1706 models. However, the display assemblies are different parts due to housing differences. They're not directly interchangeable.

Does this model have flexgate issues?

Yes, the A1708 can develop display cable wear similar to Touch Bar models. The cable routing through the hinge creates the same stress points that cause stage lighting and other symptoms.

Is A1708 the same for 2016 and 2017?

Yes, both years share the A1708 identifier with identical display assemblies. The 2017 has newer processors but screen parts are interchangeable between model years.

Is function key model easier to repair?

The repair process is similar between function key and Touch Bar models. Both require careful display assembly work. The main difference is absence of Touch Bar component.
